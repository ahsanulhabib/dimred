DIMRED software
Version 0.11		Saturday 12 Jul 2008 at 09:15

Release

Dimensional Reduction demos toolbox. A set of simple scripts for illustrating facets of dimensional reduction first written for a summer school tutorial in the area and updated for an ICML tutorial.

Version 0.11
------------

Second release with ICML tutorial examples.

Version 0.1
-----------

First release with the Data Modelling summer school examples.


MATLAB Files
------------

Matlab files associated with the toolbox are:

demIsomap.m: Try isomap on the toy data sets.
demDigitsManifold.m: Project and rotate an artificial data set of handwritten digits. 
demStickDnet2.m: Model the stick man using an GTM -- 20x20 grid.
demRotationDist.m: Small movie of rotation required to improve variance retention.
demOilDnet5.m: Oil data using DNET with 400 points.
dimredPlotMog.m: Plot a mixture of Gaussians plus data.
demCmdsSixes.m: Do classical scaling on the six data.
demSixMog1.m: Demonstrate mixture of Gaussians sixes.
demStickMog1.m: Demonstrate mixture of Gaussians on stick man.
demManifold.m: Show rotation of the digit 6/9.
demCmdsRoadData.m: This script uses classical MDS to visualise some road distance data.
dimredPrepPlot.m: Dimensional reduction plot preparation.
demStickGtm2.m: Model the stick man using an GTM -- 20x20 grid.
demManifoldPrint.m: Print the principal components of the artificial digits data set.
demStickPpca1.m: Model the stick man using Probabilistic PCA
demIcmlTalk.m: Show demos for ICML talk in order.
demSixKpca.m: Plot Kernel PCA associated with the rotated six data.
demStickDnet4.m: Model the stick man using an GTM -- 60x60 grid.
demStickDnet3.m: Model the stick man using an GTM -- 30x30 grid.
icmlFigures.m: Plot some of the figures used in ICML tutorial.
demOilDnet2.m: Oil data using GTM with 20x20 grid.
demOilFgplvm1.m: Oil data using FGPLVM with fully independent training conditional.
demSixDistances.m: Plot distance matrix associated with the rotated six data.
demOilDnet4.m: Oil data using DNET with 100 points.
generateManifoldData.m: Generates simple toy data sets for analyzing.
demOilDnet3.m: Oil data using GTM with 30x30 grid.
demStickDnet1.m: Model the stick man using an GTM 10x10 grid.
dimredToolboxes.m: Toolboxes required for the dimensional reduction demos.
demStickGtm3.m: Model the stick man using an GTM -- 40x40 grid.
demOilDnet1.m: Oil data using GTM with 10x10 grid.
demOilPpca1.m: Oil data using PPCA.
demLle.m: Try LLE on the toy data sets.
demStickGtm4.m: Model the stick man using an GTM -- 60x60 grid.
